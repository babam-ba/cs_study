# React의 컴포넌트 분리하기
> 리액트 컴포넌트를 쪼개는 기준이 있나요?⭐️⭐️⭐️

컴포넌트를 나누는 이유는 반복되는 요소들을 매번 개발하지 않고 미리 만들어둔 요소를 재사용하면 이득을 취할 수 있기 때문입니다.

모델(데이터 스키마)을 기준으로 컴포넌트를 분리하는 것이 좋다고 생각합니다. 모델이 바뀜에 따라 컴포넌트도 마찬가지로 바꿔줘야 하기 때문입니다.
