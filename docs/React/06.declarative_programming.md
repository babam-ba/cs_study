# 리액트에서 선언형 프로그래밍

> 리액트에서 선언형 프로그램은 정확히 어떤 것인가요?⭐️

## 선언형 프로그래밍 특징

## 리액트가 선언형 프로그래밍인 이유

리액트가 선언형 프로그래밍이라는 것은 JSX를 보면 알 수 있다.<br/>
JSX 자체가 선언형은 아니지만, JSX의 캡슐화를 통해 선언형 코드 작성이 가능하다.

```jsx
const App = () => {
  <Nav />
  <Menu />
  <Footer />
}
```

`<Nav />`, `<Menu />`, `<Footer />`는 JSX로 작성된 코드를 캡슐화한 컴포넌트다. 각 컴포넌트의 자세한 코드 내용은 직접 들여댜봐야 알겠지만 어떤 역할을 하는지, 어떤 순서로 렌더링 될지는 예측할 수 있다.

또한 우리는 `<Nav />`, `<Menu />`, `<Footer />`가 렌더링되는 과정까지 알 필요가 없다.
